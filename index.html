About This Project

End-to-End ML Pipeline with FastAPI Backend

This project demonstrates a complete machine learning pipeline — from raw clinical CSV datasets to a live REST API serving real-time predictions through this web interface.

Three Random Forest classifiers were trained on real medical datasets. Each model uses a scikit-learn Pipeline combining StandardScaler normalization and RandomForestClassifier, then saved with joblib as .pkl files.

FastAPI provides automatic input validation, structured JSON responses, and full Swagger documentation. This frontend connects directly to the API for live predictions.
